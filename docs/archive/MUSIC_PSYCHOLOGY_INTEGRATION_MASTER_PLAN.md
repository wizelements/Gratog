## SECTION 1: MUSIC PSYCHOLOGY FOUNDATIONS

### 1.1 Core Neurochemical Targets

#### **Dopamine (Reward/Anticipation)**
- **Triggered by:** Unexpected chord resolutions, climactic crescendos, harmonic tension + release
- **Application:** Use climax moments in songs when user completes gratitude exercises
- **Songs:** All three have emotionally charged peaks ideal for dopamine hits
- **Brainwave:** Beta (15-30 Hz) - associated with active engagement and reward anticipation

#### **Cortisol Suppression (Stress Reduction)**
- **Research Finding:** Listening to slow, meditative music (60-80 bpm) reduces cortisol by 35% in 20 minutes
- **Mechanism:** Parasympathetic nervous system activation (vagus nerve stimulation)
- **Application:** Use slow sections during meditation/reflection prompts
- **Evidence:** Classical/meditative music significantly reduces cortisol; Techno increases it

#### **Heart Rate Variability (HRV) Entrainment**
- **Target:** 60-72 bpm music syncs with resting heart rate (0.9-1.2 Hz vagal tone)
- **Effect:** Parasympathetic dominance → calm, focused, internally-reflective state
- **Your Songs:** Can extract 60-80 bpm passages for meditation moments
- **Biometric Integration:** Future: Sync music tempo to user's actual HRV via wearable

#### **Alpha Waves (8-12 Hz) - Meditative Awareness**
- **Conditions:** Emerge during calm, focused, introspective states
- **Music Characteristics:** Minor keys, consonant chords, steady tempo, minimal percussion
- **Effect:** Enhanced creativity, memory consolidation, emotional processing
- **Your Advantage:** Original songs likely contain contemplative sections perfect for this

#### **Theta Waves (4-8 Hz) - Deep Meditation/Hypnagogic State**
- **Induced by:** Very slow tempo (40-50 bpm), ambient textures, sustained tones
- **Effect:** Access to subconscious, emotional processing, memory integration
- **Application:** "Deep Gratitude" mode - remixed sections at ultra-slow tempo
- **Neurochemistry:** Increased serotonin, reduced norepinephrine (counter-stress)

#### **Gamma Waves (30-100 Hz) - Integration/Insight**
- **Triggered by:** Complex musical structures, modulation, unexpected harmonic moves
- **Effect:** Cognitive binding, insight moments, "aha" experiences
- **Application:** Use during gratitude insights/reflections

#### **Layer 7: Generative & Adaptive Remixing (The AI Orchestrator)**
**Purpose:** Prevent habituation, create infinite variation while maintaining psychological consistency

**Remix Strategies:**

A. **Temporal Remixing**
```
Take 30-60s snippets, arrange in different orders:
- Pattern A: Intro → Middle → Climax (Standard arc)

- Pattern C: Middle → Intro → Expansion (Meditation focus)

```

B. **Frequency Filtering**
```
Use parametric EQ to create "mood variations":
- Low-pass filter (treble reduction): Sleep/deep meditation mode
- High-pass filter (bass reduction): Morning energy mode
- Narrow boost: Highlight specific instruments for focus
Result: Same snippet, neurologically different effects
```

C. **Tempo Variation (Algorithmic)**
```
Time-stretch audio to create variations:
- 95% speed: Calm, contemplative (slightly slowed)
- 105% speed: Energetic, motivating (slightly quickened)
Keep pitch constant, vary energy
```

D. **Layering/Stems**
```
If extracting instrument stems:
- Piano only: Meditation mode
- Vocals only: Emotional connection
- Full mix: Standard
- Stripped (drums + bass): Groove mode
```

   * Select optimal music snippet based on psychological context
   * Maps user state → neurological target → music moment
   */
  async selectSnippet(context: MusicContext): Promise<Snippet> {

    const snippet = this.snippetDb.find(s => 
      s.neuroBand === neuralTarget.band &&
      s.emotion === neuralTarget.emotion &&
      s.arousal === neuralTarget.arousal
    );
    
    return snippet || this.fallbackSnippet(context);
  }

   * Create smooth transitions between tracks
   * Prevents jarring shifts, maintains parasympathetic tone
   */
  async transitionTo(nextSnippet: Snippet, duration: number = 3000) {
    const fadeOut = this.createFade('out', 1000);
    const fadeIn = this.createFade('in', 1000);
    
    await Promise.all([
      this.applyFade(this.currentTrack, fadeOut),
      this.preloadSnippet(nextSnippet),
    ]);
    
    this.currentTrack = nextSnippet.audioElement;
    this.applyFade(this.currentTrack, fadeIn);
    this.currentTrack.play();
  }

  /**
   * Detect user sentiment from interaction patterns
   * Maps behavior → emotional state → music response
   */
  detectUserSentiment(): 'positive' | 'struggling' | 'neutral' {
    const { typingSpeed, pauseFrequency, sessionLength } = this.sessionMetrics;
    
    if (typingSpeed > 60 && pauseFrequency < 0.3) return 'positive';
    if (pauseFrequency > 0.5) return 'struggling';
    return 'neutral';
  }

  /**
   * Neuroendocrine timeline tracking
   * Monitors psychological state progression through session
   */
  advanceNeuroPhase() {
    const elapsed = Date.now() - this.sessionMetrics.startTime;

export class SnippetSelector {
  /**
   * Intelligent snippet selection engine
   * Context aware, prevents repetition, optimizes for psychological flow
   */
  static getOptimal(context: MusicContext, excludeRecent: string[] = []): Snippet {
    const candidates = SNIPPET_DATABASE.filter(s => {
      // Must match context requirements
      if (!s.bestFor.includes(context.sessionPhase)) return false;
      
      // Avoid recent repetition (habituation prevention)
      if (excludeRecent.includes(s.id)) return false;
      
      // Respect user preferences
      const settings = useUserMusicSettings();
      if (!settings.allowedEmotions.includes(s.primaryEmotion)) return false;
      
      return true;
    });
    
    // Score candidates by appropriateness + novelty
    const scored = candidates.map(s => ({
      snippet: s,
      score: this.calculateFitScore(s, context) + this.noveltyBonus(s),
    }));
    
    return scored.sort((a, b) => b.score - a.score)[0].snippet;
  }

  private static calculateFitScore(snippet: Snippet, context: MusicContext): number {
    const emotionMatch = context.userEmotion === snippet.primaryEmotion ? 10 : 0;
    const arousalMatch = Math.abs(
      AROUSAL_LEVELS.indexOf(context.userArousal) - 
      AROUSAL_LEVELS.indexOf(snippet.arousalLevel)
    ) === 0 ? 5 : 0;
    const timeMatch = this.isTimeOfDayOptimal(snippet) ? 3 : 0;
    
    return emotionMatch + arousalMatch + timeMatch;
  }

  private static noveltyBonus(snippet: Snippet): number {
    const timeSinceLastPlay = Date.now() - (snippet.lastPlayedAt || 0);
    const hoursElapsed = timeSinceLastPlay / (1000 * 60 * 60);
    
    // Bonus increases with time since last play (up to 6 points after 24+ hours)
    return Math.min(6, hoursElapsed / 4);
  }
}

export class AudioManager {
  private context: AudioContext;
  private masterGain: GainNode;
  private analyser: AnalyserNode;
  
  constructor() {
    const AudioContext = window.AudioContext || window.webkitAudioContext;
    this.context = new AudioContext();
    
    // Create audio graph
    this.masterGain = this.context.createGain();
    this.analyser = this.context.createAnalyser();
    
    this.masterGain.connect(this.analyser);
    this.analyser.connect(this.context.destination);
  }
  
  /**
   * Load and play audio with automatic crossfade
   */
  async playSnippet(
    audioPath: string,
    options: PlayOptions = {}
  ): Promise<void> {
    const {
      volume = -10,      // dB
      fadeInDuration = 1000, // ms
      fadeOutDuration = 1000,
    } = options;
    
    const audioElement = await this.loadAudio(audioPath);
    const source = this.context.createMediaElementAudioSource(audioElement);
    source.connect(this.masterGain);
    
    // Fade in
    this.masterGain.gain.setValueAtTime(0, this.context.currentTime);
    this.masterGain.gain.linearRampToValueAtTime(
      this.dbToLinear(volume),
      this.context.currentTime + fadeInDuration / 1000
    );
    
    audioElement.play();
  }
  
  async transitionTo(nextPath: string, crossfadeDuration: number = 3000) {
    // Fade out current
    this.masterGain.gain.linearRampToValueAtTime(
      0,
      this.context.currentTime + crossfadeDuration / 2000
    );
    
    // Load next
    const nextElement = await this.loadAudio(nextPath);
    const nextSource = this.context.createMediaElementAudioSource(nextElement);
    nextSource.connect(this.masterGain);
    
    // Fade in next
    setTimeout(() => {
      nextElement.play();
      this.masterGain.gain.linearRampToValueAtTime(
        this.dbToLinear(-10),
        this.context.currentTime + crossfadeDuration / 2000
      );
    }, crossfadeDuration / 2);
  }
  
  /**
   * Get real-time frequency data for visualization
   */
  getFrequencyData(): Uint8Array {
    const dataArray = new Uint8Array(this.analyser.frequencyBinCount);
    this.analyser.getByteFrequencyData(dataArray);
    return dataArray;
  }
  
  private dbToLinear(db: number): number {
    return Math.pow(10, db / 20);
  }

      // Load full quality MP3
      return this.playSnippet(snippets[snippetId].highQualityUrl);
    } else if (connection?.effectiveType === '3g') {
      // Load compressed MP3
      return this.playSnippet(snippets[snippetId].mediumQualityUrl);
    } else {
      // Load low-bandwidth version
      return this.playSnippet(snippets[snippetId].lowQualityUrl);
    }
  }
}
