## 📞 Troubleshooting

**Problem:** No sound on page load  
**Solution:** Check browser console → See AUDIO_SPECS.md Error Handling section

**Problem:** Audio pops/clicks  
**Solution:** Verify fade times > 50ms → See AUDIO_SPECS.md Crossfade section

**Problem:** iOS silent toggle mutes audio  
**Solution:** Respect iOS restrictions → See AUDIO_SPECS.md Mobile section

**Problem:** Audio files 404  
**Solution:** Verify file paths in snippetDatabase.ts → See QUICK_START.md STEP 3

**Problem:** Memory leak (audio keeps playing)  
**Solution:** Implement cleanup in useEffect → See QUICK_START.md code example

For other issues:
1. Check document table of contents
2. Search for keyword (Ctrl+F)
3. Refer to relevant section
4. Check code examples
