CORE FEATURES
================================================================================

1. INTELLIGENT SNIPPET SELECTION
   - 12 snippets total (4 per song)
   - Mapped to 5 session phases (intro, reflection, journal, share, meditation)
   - Emotion filtering (peace, hope, vulnerability, acceptance, joy)
   - Arousal levels (low, medium, high)
   - Prevents repetition (tracks last 3 selections)
   - Fallback mechanism if no candidates

2. AUDIO PLAYBACK
   - Native Web Audio API (no external libraries)
   - Smooth fade in/out animations (2 second default)
   - 50ms update intervals for animation smoothness
   - Lazy loading on demand (not preloaded)
   - Memory cleanup on unmount

3. VOLUME CONTROL
   - Logarithmic dB scale (-20 to 0)
   - Real-time adjustment
   - Smooth volume transitions
   - Visual feedback in UI

4. USER PREFERENCES
   - LocalStorage persistence
   - Saves volume setting (music_volume)
   - Saves enable/disable state (music_enabled)
   - Survives page reloads and browser restarts

5. GLOBAL STATE
   - React Context API (no Redux, no Zustand)
   - Single source of truth
   - useMusic() hook for easy consumption
   - Immutable updates with setState

6. UI WIDGET
   - Fixed bottom-right corner positioning (z-50)
   - 🎵 toggle button
   - Expandable control panel
   - Responsive design
   - Dark mode support
   - Keyboard navigable
   - ARIA compliant
