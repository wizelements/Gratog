## 📋 COPYWRITING PHILOSOPHY

**Before**: Functional, descriptive copy  
**After**: Benefit-driven, trust-building, emotionally resonant copy

**Core Principles**:
1. Lead with benefits, not features
2. Build trust through specificity ("Atlanta-made" > "locally made")
3. Create urgency without being pushy
4. Make wellness aspirational yet accessible
5. Use sensory language ("smooth", "rich", "vibrant")
